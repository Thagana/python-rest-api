# python-rest-api
Rest api using python flask and sqlite3
## End Points
## GET
http://localhost/api/v1/movies
Get all movies
## GET
http://localhost/api/v1/movie/1
Get movie with id 1
## DELETE
http://localhost/api/v1/movies/1
Delete movie with id 1
## PUT
http://localhost/api/v1/movies/1
Update movie with id 1
